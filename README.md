# My-first-website

🛒 Integrated E-commerce Store that provides a perfect user experience for shopping and purchasing 💳

🔗 Store Link
https://myfirstwebsitestore.netlify.app/

⚙️ Technologies Used

- Front-End: React, MUI, Axios, Swiper
- Back-End: Node.js, Express.js
- Database: PostgreSQL, Render, Neon
- 🔐 Authentication: JWT

🌟 Project Features

- 📝 Registration and Login System
- 🔍 Browse products with filtering
- 📦 Simple order tracking system
- 💰 Payment system
- 🛍️ Shopping cart to add product
